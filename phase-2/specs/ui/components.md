# UI Components

## Core Components

### `TaskList`

- **Display**: Card-based layout with hover effects.
- **Icons**: Uses `lucide-react` for actions (Edit, Delete, Check).
- **Edit Mode**: Inline editing for Title and Description.
- **Filtering**: Filters for All/Pending/Completed.

### `Auth`

- **Forms**: Email/Password Sign In & Sign Up.
- **Social Login**: Buttons for Google and GitHub.
- **Design**: Modern card layout with gradient textual accents.

### `Layout`

- Navigation bar with app branding and user session info.
- Responsive container for all screen sizes.
