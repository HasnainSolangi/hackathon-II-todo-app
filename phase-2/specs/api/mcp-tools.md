# MCP Tools (Planned)

## Overview

Definition of Model Context Protocol (MCP) tools exposed to the AI agent.

## Tools

### `create_task`

- **Description**: Creates a new task.
- **Arguments**: `title` (string), `description` (string, optional).

### `list_tasks`

- **Description**: Lists tasks for the user.
- **Arguments**: `status` (string, optional).

### `complete_task`

- **Description**: Marks a task as complete.
- **Arguments**: `task_id` (integer).
