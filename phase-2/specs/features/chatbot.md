# Feature: AI Chatbot (Planned)

## Overview

An AI-powered chatbot interface for managing todos via natural language.

## User Stories

- As a user, I can add tasks by typing "Remind me to..."
- As a user, I can ask "What do I have to do today?"
- As a user, I can mark tasks complete via chat.

## Technical Requirements

- **LLM**: OpenAI / Anthropic via MCP.
- **Context**: Chatbot has access to User's Task List.
- **Interface**: Chat bubble UI in the web app.
